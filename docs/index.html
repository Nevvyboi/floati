Data Storage

All data you enter into Floati (expenses, budgets, subscriptions, notes, reminders, receipts) is stored locally on your device in a SQLite database file. This file lives in your user profile directory and is never uploaded, synced, or transmitted anywhere.

What We Don't Collect

  • Personal information (name, email, address)
  • Financial data (bank accounts, card numbers)
  • Usage analytics or telemetry
  • Crash reports or diagnostics
  • Device identifiers or fingerprints
  • Location data
  • Cookies or tracking pixels

No Internet Required

Floati works entirely offline. The application does not make any network requests, connect to any servers, or require an internet connection to function.

Data Export & Portability

You can export all your data at any time as JSON, PDF, Excel, or CSV files. You own your data completely. You can delete all data by removing the database file from your device.

Third-Party Services

Floati does not integrate with or send data to any third-party services, advertising networks, analytics platforms, or social media platforms.

Children's Privacy

Since Floati does not collect any data from any user, it does not knowingly collect data from children under 13.

Changes

If this privacy policy changes, the updated version will be published at this URL. Since Floati collects no data, meaningful changes are unlikely.
